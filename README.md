# Prometheus Web Dashboard

Prometheus Web is a property management dashboard where administrators can control properties, tenants, leases, and payments from a single interface. The app centralizes entity data, simplifies CRUD operations, and provides modals and data tables to keep day-to-day rental workflows organized.

## Quick Start

Install dependencies and start the development server:

```bash
npm install
npm start
```

The app runs in development mode at [http://localhost:3000](http://localhost:3000) with automatic reloads while you edit.
